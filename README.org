#+title: Wire-Cell Toolkit ZeroMQ/protobuf interface to ZIO

This package provides the ~WireCellZpb~ plugin library to the Wire-Cell
Toolkit.  It gives a mechanism to extract or inject data between an
executing WCT graph and external entities.  The transport is provided
using [[https://github.com/brettviren/zio][ZIO]] (functionality on top of ZeroMQ technologies) and the data
WCT is serialized as protobuf objects.

This package is built using WCT's ~wcb~ Waf-based build tool.

#+begin_example
  $ wcb configure --prefix=/path/to/install
  $ wcb --notests install
  $ wcb --alltests
#+end_example

External dependencies, including Wire-Cell Toolkit may be
auto-detected or you may provide hints.  Use ~wcb --help~ for details.

Like all WCT plugins you can dump the default configurations of the
components ZPB provides with:

#+begin_example
  $ wire-cell -a ConfigDumper -p WireCellApps -p WireCellZpb| json_pp
#+end_example
